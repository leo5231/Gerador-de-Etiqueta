Sistema de geração de etiquetas

Ferramenta para acelerar a identificação de produtos no mostruário, com foco em leitura por bipagem e ganho de produtividade na contagem.

Como funciona

O usuário coleta os SKUs no GAN, limpa os dados para remover informações desnecessárias e depois gera etiquetas com código de barras para impressão em PDF.

  1. Acesse o Limpador e cole o texto vindo do GAN.
  2. Copie o resultado limpo no formato correto.
  3. No Gerador, clique em Buscar para montar as etiquetas.
  4. Clique em Gerar PDF para imprimir ou salvar.
